# 🌱 FreeMe — מסע שיפור עצמי

אפליקציה אישית PWA להתמודדות עם התמכרות לפורנוגרפיה, בנויה על Gemini AI.

---

## 🚀 התקנה בטלפון

### שלב 1: העלה ל-GitHub Pages

1. צור repository חדש ב-GitHub (שם לדוגמה: `freeme`)
2. העלה את כל הקבצים (drag & drop בממשק GitHub)
3. לך ל-**Settings → Pages → Source: Deploy from a branch → Branch: main**
4. המתן כ-2 דקות — האפליקציה תהיה זמינה ב: `https://[USERNAME].github.io/freeme`

### שלב 2: שמור כאפליקציה בטלפון

**iOS (Safari):**
1. פתח את הקישור ב-Safari
2. לחץ על כפתור השיתוף ↑
3. בחר "הוסף למסך הבית"

**Android (Chrome):**
1. פתח את הקישור ב-Chrome
2. לחץ על 3 הנקודות ⋮
3. בחר "הוסף למסך הבית"
